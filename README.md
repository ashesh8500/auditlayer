# AuditLayer

**Social media competitive intelligence for evidence-based creators.**

Enter a handle, answer 3 questions, get a self-contained HTML audit report: where you're at, what's holding you back, who's doing it better, what to post next week, when you'll hit the next milestone, and the money move.

**Founders:** Ashesh Kaji (tech/infra) + Narin (domain, calibration, clients)

---

## v2 stack (current)

| Layer | Path | Host |
|---|---|---|
| Web portal | `web/` | Vercel |
| Control plane | `supabase/` | Supabase Cloud |
| Report worker | `worker/` | Long-lived host (Hetzner / laptop) |
| Legacy v1 | `legacy/` | Archived — do not extend |

```
Browser → Next.js (web/) → Supabase (Auth, Postgres, Storage, Realtime)
                                ↑
                           worker/ + Hermes Gateway → self-contained HTML
```

**Prod (current):** https://web-delta-dun-29.vercel.app

## Quick start (agents & devs)

```bash

# Local web
make dev-web

# Worker (needs worker/.env — see worker/.env.example)
make worker-run
```
