# Pulse Audit — Landing Page Pattern

The Pulse Audit is the free entry product on auditlayermedia.com. It uses the Compact Brand Pulse format (5 sections: score bars, strengths, gaps, moves, weekly rhythm) and requires only an email — no credit card.

## Landing page section order

Hero → Full Report Mock (11-section preview) → Six Questions → **Pulse Audit** → Pricing → Blueprint → FAQ

## Pulse Audit copy

- Section heading: "Free Pulse Audit"
- Subtitle: "All you need is an email · 2 free runs · No credit card"
- Mock report header bar: "Brand Pulse · Sample Report" with green "Generated" badge
- Mock handle: @solawellness (clean sunscreen + skincare)
- Teaser: "⚡ Upgrade for the full report" (minimal — one line, no feature list)
- CTA button: "Run Your First Pulse Audit"
- Below CTA: "After your two free Pulse audits, upgrade to a plan to keep going."

## How the upsell works

1. User enters email → gets 2 free Pulse Audit runs
2. Each run produces a 5-section compact Brand Pulse report
3. After 2 runs, upgrade required → links to Pricing section (Starter/Pro/Enterprise)
4. The full report mock above the Pulse section shows what upgrading unlocks
