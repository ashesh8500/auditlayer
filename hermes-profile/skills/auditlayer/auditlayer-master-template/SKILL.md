# AuditLayer Master Report Template

This is the **single source of truth** for all AuditLayer reports. Every audit generated by the worker MUST follow this exact 15-section structure, CSS token system, and component patterns. The canonical reference implementation is the @shaimastrategist report on Supabase — when in doubt, that report's format wins over any description here.

## Quick Reference

**15 Sections:**
1. Executive Summary
2. Key Metrics
3. Strengths
4. Weaknesses
5. Root Cause Analysis
6. Peer Comparison
7. Content Format Analysis
8. Engagement Growth Strategy
9. Content Calendar & Creative Board
10. Quick Wins — This Week
11. Success Benchmarks
12. Audience Profile
13. Road to Milestone
14. Audit Cadence
15. Get the Execution Plan

### Tables
```css
.data-table {
  width: 100%;
  border-collapse: separate;   /* NOT collapse */
  border-spacing: 0;
  border: 1px solid var(--line);
  border-radius: var(--radius-sm);
  overflow: hidden;
  box-shadow: var(--shadow-sm);
}
.data-table th {
  padding: 12px 16px;
  background: #fafaf8;
  font-size: 0.7rem; font-weight: 600; text-transform: uppercase; letter-spacing: 0.08em;
}
.data-table td {
  padding: 12px 16px;
  background: var(--surface);

.data-table tr:last-child td { border-bottom: none; }
.data-table tr.highlight td { background: var(--accent-muted); }
```

**DO NOT** wrap tables in `<div class="table-wrap">` — tables handle their own overflow via `border-radius` + `overflow: hidden`.

### Tags
```css
.tag {
  display: inline-block; font-size: 0.7rem; font-weight: 600;
```

Tags are PILL-shaped (`border-radius: 12px`) with rounded full caps — NOT squared-off rectangles.


`.callout.warning`: amber background + left amber border (`var(--amber-muted)` / `var(--amber)`)

## Template Variables

Per-audit swap:
- `{handle}` — Instagram handle
- `{platform}` — Instagram / TikTok / YouTube
- `{date}` — report date
- `{follower_count}` — current followers
- `{following_count}` — current following
- `{post_count}` — total posts
- `{ff_ratio}` — following/follower ratio
- `{niche}` — account vertical
- `{goal}` — Rebrand / Growth / Launch / Monetize
- `{milestone}` — 2K / 5K / 10K / 50K etc
- `{peer_a/b/c_followers}` — peer follower counts
- `{content_pillars}` — 3-5 pillar names for creative board

## Rules

- Peer followers MUST be higher than subject
- Zero Twitter/X references anywhere
- No methodology footer section

- All CSS inline — zero external dependencies
- Light theme only — this is client-facing
